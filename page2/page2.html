Why the interest in Software Engineering?

I always thought the concept of creating things from nothing but code was always interesting to me, I've played tons of games where people used their own code to create content in the form of mods to games and the only problem I had with that is that eventually that modder would eventulaly quit the game or stop coding for whatever reason, and they'd leave their mod's source code for anyone to pick up and use. But being me and being inexperienced in coding I wouldn't know what to do with the code and what any of it even mean't. So I decided to pursue an education in Comp Sci to later become a software engineer, After some studying I realized that programming and coding had many applications besides mods for games, Some people code hardware to do specific things like monitor heartbeats or to trigger something when a sensor senses something. I thought it was really cool to learn about all these different applications of coding so I decided to stick to it as my current Major.
